
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Create a Hangman game where players guess letters to reveal a hidden word before running out of attempts. You will practice string manipulation, loops, conditionals, and random selection.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description
Implement a complete Hangman game using Python that guides players through multiple rounds of guessing letters to reveal a hidden word before running out of attempts.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and display current progress (e.g., `_ _ _` format)
- Track incorrect guesses remaining and display them to the player
- End the game when the word is guessed or attempts are exhausted
- Display appropriate win/lose messages with final game state
